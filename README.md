# Sample project with Kekkonen

Running at: https://kekkonen.herokuapp.com/

Kekkonen: http://kekkonen.io

## Usage

### Run the application locally

```
lein repl
user=> (go)
```

### Packaging and running as standalone jar

```
lein uberjar
java -server -jar target/example.jar
```
